このアプリについて知る

このアプリでできる事

サイズの変更や枠線の追加など画像の加工を行うためのアプリです。

ファイルをアップロードすると変換後の画像が即ダウンロードできます。複雑な画像加工はできませんが、大量のファイルに簡単な加工を繰り返し施すのに向いています。

アプリの利用について

URLにアクセスするだけで誰でも簡単に利用する事ができます。

  • サインアップしてアカウント登録する必要はありません。
  • ダウンロード回数の制限はありません。

画像の加工には Canvas APIを利用しています。ファイルサイズの大きな画像を加工した際に画質が劣化する事がありますので、加工前のオリジナルのファイルはご自身で保管をお願いいたします。

お願い

このサイトへのアクセスは GoogleAnalytics を使って情報収集されます。目的は作者がコンバージョン設定など GoogleAnalytics の使い方を学ぶためです。どうかご協力をお願いいたします。分析したデータを公開するような事はありませんのでご安心ください。
